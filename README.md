
# 🚀 API Hunter

**API Hunter** is a professional React-based web application designed to demonstrate state management for asynchronous API interactions. It provides a seamless interface for users to execute `GET` and `POST` requests, offering a side-by-side comparison of the **native Fetch API** and the **Axios library**.

## 🎯 Key Features

- **Global State Management:** Efficiently tracks states for loading, success, and error using Redux Toolkit.
- **Dual-Library Support:** Integrated implementation of both Fetch API and Axios for HTTP requests.
- **Dynamic Request Builder:** A user-friendly form to input custom endpoints, methods, and JSON body data.
- **Real-Time Response Summary:** Displays a detailed breakdown of the API response, including the method used, the endpoint hit, and formatted JSON output.
- **Error Handling:** Robust error catching for network issues, 404 Not Found, and bad status codes.

## 🛠️ Technology Stack

- **Frontend:** React (Vite)
- **State Management:** Redux Toolkit (RTK)
- **API Clients:** Axios & Fetch API
- **Styling:** Modern CSS / Tailwind CSS
- **Local Database:** JSON Server (for mock backend testing)
